Weather Meister

Problem Statement

This is a class project for CSCI 201: Software Development.

WeatherMeister allows users to search for weather at any location in world. The application contains a search bar with auto-complete feature as well as the ability to create an account. Historical searches are also saved (if the user is logged in). Never forget to bring an umbrella and use WeatherMeister!



Tools:

  • Languages: HTML, CSS, JavaScript, Java
  • APIs: OpenWeatherMap
  • IDE: Eclipse
  • Acceptance Tests: Cucumber
